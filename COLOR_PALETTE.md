# Цветовая палитра сайта

## Выбранная схема: Технологичный минимализм с бирюзовыми акцентами

### Обоснование выбора

**Teal/Cyan (#14b8a6)** как основной акцентный цвет выбран по следующим причинам:

1. **Психология цвета:**
   - Бирюзовый ассоциируется с технологиями, инновациями и точностью
   - Создает современную, профессиональную атмосферу
   - Отлично сочетается с темным фоном hero-секции
   - Подходит для tech-сервиса - внушает доверие и экспертность

2. **UI/UX преимущества:**
   - Идеальная контрастность с темно-синим фоном (#0f172a)
   - Современный tech-бренд внешний вид
   - Хорошо работает на всех устройствах
   - Проходит WCAG AA стандарт контрастности

3. **Для минималистичного дизайна:**
   - Гармонирует с hero-bg изображением (HACKLAB)
   - Строгий и элегантный, tech-стиль
   - Подчеркивает важные элементы без избыточности

## Основные цвета

### Primary (Основной акцентный)
```css
--primary: #052533          /* Кнопки и основные акценты */
--primary-hover: #041d28    /* Hover состояния кнопок */
--primary-light: #99f6e4    /* Teal 200 - светлые акценты */
--primary-dark: #0f766e     /* Teal 700 - темные варианты */
```

**Где использовать:**
- Основные CTA кнопки (#052533)
- Ссылки и активные элементы
- Иконки и акценты
- Hover эффекты на заголовках

**Примеры:**
```tsx
<Button>Смотреть все статьи</Button>
<Link className="text-primary hover:text-primary-hover">...</Link>
<Icon className="text-primary" />
```

---

### Secondary (Вторичный - фоны)
```css
--secondary: #0f172a         /* Slate 900 - темно-синий фон */
--secondary-light: #1e293b   /* Slate 800 - более светлый */
--secondary-lighter: #334155 /* Slate 700 - еще светлее */
```

**Где использовать:**
- Hero section фон
- Header, footer
- Темные секции
- Контрастные блоки

**Примеры:**
```tsx
<section className="bg-secondary text-white">...</section>
<header className="bg-secondary-light">...</header>
```

---

### Accent (Удален из палитры)
**Примечание:** Дополнительный акцентный цвет удален для соблюдения минималистичного дизайна.
Используйте только `primary` (Amber) и серые оттенки.

### Success (Успех)
```css
--success: #10b981          /* Emerald 500 */
--success-light: #d1fae5    /* Emerald 50 */
```

**Где использовать:**
- Подтверждения действий
- Успешные уведомления
- Статусы "выполнено"

**Примеры:**
```tsx

```

## Тени с цветовыми тонами

```css
shadow-primary-glow: 0 0 20px rgba(245, 158, 11, 0.2)
shadow-primary-sm: 0 2px 8px rgba(245, 158, 11, 0.1)
shadow-primary-md: 0 4px 16px rgba(245, 158, 11, 0.15)
```

**Где использовать:**
```tsx
<Card className="hover:shadow-primary-md">...</Card>
<Button className="shadow-primary-sm">...</Button>
```

**Примечание:** Для строгого минимализма используйте тени умеренно, предпочитайте серые тени для большинства элементов.

## Доступность (a11y)

Все цветовые комбинации проверены на контрастность:

- `primary` (#f59e0b) на белом фоне: **2.92:1** (подходит для крупных элементов, кнопок, иконок)
- `primary-dark` (#b45309) на белом фоне: **5.24:1** (AAA для крупного текста, AA для обычного)
- Белый текст на `primary`: **2.92:1** (подходит для крупного текста в кнопках)
- Белый текст на `secondary`: **15.33:1** (AAA)

**Рекомендация:** Для основного текста используйте серые оттенки. Amber используйте для акцентов, кнопок и заголовков.

## Миграция цветовой схемы

### Было → Стало

| Старый цвет | Новый цвет | Контекст |
|-------------|------------|----------|
| `#0ea5e9` (sky blue) | `#f59e0b` (amber) | Primary акценты |
| `#0284c7` (sky hover) | `#d97706` (amber hover) | Hover эффекты |
| `text-blue-600` | `text-primary` (amber) | Акценты и ссылки |
| `bg-blue-50` | `bg-amber-50` | Светлые фоны |

## Примеры компонентов

### Кнопка с иконкой
```tsx
<Button className="group">
  Смотреть все статьи
  <ArrowRight className="ml-2 group-hover:translate-x-1 transition-transform" />
</Button>
```

### Карточка статьи
```tsx
<Card className="hover:shadow-primary-md transition-all duration-350">
  <h3 className="group-hover:text-primary transition-colors duration-250">
    Заголовок
  </h3>
</Card>
```

### Интерактивная ссылка
```tsx
<Link className="flex items-center gap-2 text-white hover:text-primary transition-colors duration-250">
  <Phone className="w-5 h-5" />
  <span>+420 607 855 558</span>
</Link>
```

## Дальнейшие рекомендации

1. **Для акций/скидок** используйте `primary` (amber) для выделения
2. **Для предупреждений** используйте `primary-dark` (#b45309)
3. **Для ошибок** используйте красный, но приглушенный (#ef4444 - red-500)
4. **Строгий минимализм**: избегайте избыточных цветовых акцентов, используйте серый + amber
